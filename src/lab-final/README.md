# 任务

利用S5PV210的中断、串口以及定时器实现以下功能：

1. PC机通过串口向PV210发送字符串“T+play”，则PC端得到回显“Playing music”，同时PV210利用PWM定时器播放一段不少于10s的音乐（音乐自选，时间不足扣分）；PC机通过串口向PV210发送T+序号的数字（比如T+09，一共4个字母），则PC端得到回显“It is me！”；PC机通过串口向PV210发送其他“T+”的字符串，则PC端得到回显“error！”。
2. 由开发板按下SW4键，利用串口在PC端显示 “Marquee”，同时，4颗流水灯自左向右一次，第1~4颗LED亮灭时间间隔分别为0.5，1，2，2秒（要求定时器实现精准的0.5，1，2，2秒），最后全亮。
3. 要求写到一个程序中，且不用关机重启，可以一并实现上述2个题目的任务。在LED运行过程中，PC串口输入“play”，可以立刻关闭LED运行，实现音乐；在音乐播放的时候，按SW4键，4颗精准流水灯自左向右一次。
